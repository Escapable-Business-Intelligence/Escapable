## 11.10 회의록
금일 회의는 지금까지 구현된 내용에 대한 진행상황 공유와    
웹 페이지 구성 및 진행계획에 대한 내용을 회의했습니다.


## 전체 회의
- 설문조사 페이지 진행상황
    - 데이터베이스 연결하기
    - 카페 테이블 정보 불러오기
    - 간단하게 UI 제작하기 (간단한 설문조사, 복잡한 설문조사)
- 예측 결과값 평가 방법
- 외부DB 연결 및 연동
    - aws 사용
    - 웹과도 연결완료
- 데이터크롤링 재시도
    - 데이터 사용시 오류가 있어 재시도함
    - 완료 후 aws서버에 배포 예정
- 앞으로의 계획
    - 데이터 업로드 -> 설문조사 페이지 완료 -> 머신러닝 연동 후 결과 데이터 전송 -> 결과 페이지, 테마 상세보기 페이지
